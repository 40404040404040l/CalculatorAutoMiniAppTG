# CalculatorAutoMiniAppTG

**CalculatorAutoMiniAppTG** — статический Telegram Mini App для расчета примерной стоимости автомобиля под ключ с учетом таможенной пошлины, утильсбора, доставки, дополнительных расходов и курса валюты.

Проект сделан без backend: вся логика находится в браузере, а формулы перенесены из Excel-файла `Калькулятор-2.xlsx` в `miniapp/app.js`.

## Что умеет приложение

- принимать основные параметры автомобиля;
- рассчитывать таможенную пошлину;
- рассчитывать утильсбор;
- учитывать брокеров, стоянку, перевод денег, оплату с фирмы и доставку;
- показывать цену под ключ в валюте;
- показывать итоговую стоимость в рублях;
- работать как обычная web-страница и как Telegram Mini App.

## Входные параметры

В интерфейсе используются:

- цена с вычетом НДС;
- объем двигателя в см3;
- год выпуска автомобиля;
- мощность в лошадиных силах;
- брокеры, стоянка и прочие расходы;
- перевод денег и оплата с фирмы;
- доставка до Санкт-Петербурга или Москвы;
- курс валюты.

## Локальный запуск

Из корня проекта:

```bash
python3 -m http.server 8080
```

Откройте в браузере:

```text
http://localhost:8080/miniapp/
```

Также можно открыть `miniapp/index.html` напрямую, но запуск через локальный сервер ближе к реальным условиям Telegram Mini App.

## Деплой

Проект можно разместить на любом статическом hosting:

- Vercel;
- Netlify;
- GitHub Pages;
- Timeweb/обычный web server;
- любой Nginx/Apache со статическими файлами.

Для Vercel:

1. Создайте новый проект.
2. Подключите этот репозиторий.
3. Оставьте настройки статического проекта по умолчанию.
4. После деплоя Mini App будет доступен по URL вида:

```text
https://your-project.vercel.app/miniapp/
```

## Подключение к Telegram

1. Откройте `@BotFather`.
2. Создайте бота через `/newbot` или выберите существующего.
3. Перейдите в `/mybots`.
4. Выберите нужного бота.
5. Откройте `Bot Settings`.
6. Выберите `Menu Button`.
7. Укажите публичный HTTPS URL приложения.
8. Откройте бота и нажмите кнопку меню.

Telegram Mini Apps требуют публичный HTTPS-адрес. Локальный `http://localhost` подходит только для разработки и проверки в браузере.

## Как работает расчет

`miniapp/app.js`:

- считывает значения из полей формы;
- приводит ввод к числам;
- рассчитывает пошлину по возрасту автомобиля, цене и объему двигателя;
- рассчитывает утильсбор по таблицам для машин младше 3 лет и старше/равных 3 годам;
- пересчитывает итог при каждом изменении поля;
- вызывает `Telegram.WebApp.ready()` и `Telegram.WebApp.expand()`, если страница открыта внутри Telegram.

## Важно

- Расчет является ориентировочным и зависит от актуальности формул.
- При изменении правил, ставок или коэффициентов нужно обновить таблицы и функции в `miniapp/app.js`.
- Приложение не хранит данные пользователя и не отправляет их на сервер.
