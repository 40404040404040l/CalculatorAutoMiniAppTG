# Telegram Mini App: Таможенный калькулятор

## Что уже сделано
- Создана бесплатная версия Mini App без сервера.
- Формулы из `Калькулятор-2.xlsx` перенесены в `miniapp/app.js`.

## Локальный запуск
1. Открой терминал в корне проекта.
2. Запусти простой сервер:
   - `python3 -m http.server 8080`
3. Открой в браузере:
   - `http://localhost:8080/miniapp/`

## Деплой бесплатно (Vercel)
1. Зарегистрируйся на https://vercel.com
2. Создай `New Project` и импортируй эту папку (или репозиторий).
3. Для статического проекта настройки по умолчанию подходят.
4. После деплоя получишь URL вида:
   - `https://your-project.vercel.app/miniapp/`

## Подключение в Telegram
1. Открой `@BotFather`.
2. `/newbot` (если бот еще не создан).
3. `/mybots` -> выбери бота.
4. `Bot Settings` -> `Menu Button` -> вставь URL Mini App.
5. Открой бота и нажми кнопку меню.

## Структура
- `miniapp/index.html` — интерфейс
- `miniapp/styles.css` — стили
- `miniapp/app.js` — формулы и расчет
